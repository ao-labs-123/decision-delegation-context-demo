# C : 判断委ね系（Decision Delegation）
判断委ね系とは、発話者が明示的な判断や主張を避け、
意思決定・責任・規範の解釈を相手に委ねる文脈である。

## 下位分類

### C1: 立場依存
発話者と受話者の立場差によって、判断の重みが移動する文脈。

例：
- 「私が決めていいことなのか分からないですが…」

### C2: 暗黙ルール
明文化されていない慣習・空気・前例に判断を委ねる文脈。

例：
- 「いつもはどうしてましたっけ？」

### C3: 責任の所在
決定そのものではなく、責任の帰属を相手に移す文脈。

例：
- 「最終的には〇〇さんの判断になりますよね」
